# Ray Tracer

Ray Tracing Project for CS130

## Scripts

Available scripts:

### `scons`

Compiles ray-tracing source files

### `./ray_tracer -i ./N.txt`

Runs test N (00-29)

### `./ray_tracer -i ./N.txt -s ./N.png`

Compare test N (00-29) with expected results and returns a diff
